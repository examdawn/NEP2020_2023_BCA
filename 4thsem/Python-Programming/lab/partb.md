#### Bar Chart

```python
import matplotlib.pyplot as plt


courses = ["BCA","BCOM", "BBA", "BSc", "BA" ]
count = [120, 180, 60, 30, 80]
plt.bar(courses, count, color="maroon", width=0.4)

plt.xlabel("Courses")
plt.ylabel("Count")
plt.title("Count of Students in Different Courses")

plt.show()
```
